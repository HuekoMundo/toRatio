Способ 1.
	- Получим корзину, по событию добавления товара в корзину или изменения корзины(удаление/количество). 
    - выполняем функцию* для высчитывания скидки на весь заказ.
	- Создаем скидку на посчитанную ранее сумму. Создаем скидку с купонами, создаем одноразовый купон и активируем его юзеру.

Способ 2.
	- Создаем элемент инфоблока со свойствами: Цена за товар и Число, отображающее значение "каждый N-й товар".
	- При событии изменения корзины, добавление товара в корзину, удаление товара из корзины
		выполняем функцию** поиска необходимых товаров для изменения у них значения DISCOUNT_PRICE
		
*
функция примерный алгоритм
	собираем массив из всех товаров учитывая количество каждого
	array = []
	for(i=0,i<QUANTITY,i++){ //по каждому товару
		array[] = товар
	} 
	вычисляем количество товаров на которые надо применить скидку - это целочисленное деление count(array) на N
	проходим с начала массива количество раз, высчитанное в предыдущем шаге
	и вычисляем скидку на заказ += цена товара  - цена скидки
	
**
во втором случае скидка высчитывается инидивидуально для каждого товара и записывается в поле DISCOUNT
